<h4 id="3-presentation-formats">3. Presentation formats<a class="content-header" href="#3-presentation-formats">
<span class="content-header-link"><svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 20 20" fill="currentColor" class="w-5 h-5 linkicon">
        <path d="M12.232 4.232a2.5 2.5 0 0 1 3.536 3.536l-1.225 1.224a.75.75 0 0 0 1.061 1.06l1.224-1.224a4 4 0 0 0-5.656-5.656l-3 3a4 4 0 0 0 .225 5.865.75.75 0 0 0 .977-1.138 2.5 2.5 0 0 1-.142-3.667l3-3Z"></path>

<p><strong>Accept (Oral)</strong>: 15 minutes talk, not including questions. Slide format is 16:9. You can present from your laptop, or ours, in the latter case the slides must be in pdf and you must share them with us before the opening remarks. You can optionally display a poster as well, see below.</p>
<p><strong>Accept (Poster)</strong>: Per <a href="https://iclr.cc/Conferences/2025/PosterInstructions" rel="nofollow" target="_blank">official ICLR rules</a>, the maximum size allowed is 61cm x 91cm (24in x 36in) width/height, portrait format. We the organizers guarantee that there is enough space for every conforming poster. If you bring a non-conforming poster, we won't reject it outright, but you assume the following risk: should the room run out of space, we will take down the rule-breaking posters starting from the largest and least aesthetically pleasant.</p>
<p><em>You will have an opportunity to give a 1 minute spotlight talk as well.</em> Given the time constraints of the format, we will assemble all slides into a single presentation. To be included, you <strong>must upload your single 16:9 slide in pdf format to the corresponding "Spotlight" entry in the <a href="https://iclr.cc/virtual/2025/workshop/23967" rel="nofollow" target="_blank">iclr.cc workshop agenda</a> before 09:00 April 26th Singapore time.</strong></p>
<p><strong>iclr.cc</strong> Please make sure the information in the <a href="https://iclr.cc/virtual/2025/workshop/23967" rel="nofollow" target="_blank">iclr.cc workshop agenda</a> is correct, update if necessary.</p>
<p><strong>Remote participation</strong>: The workshop will be streamed via <a href="https://iclr.cc/" rel="nofollow" target="_blank">iclr.cc</a>. If you are unable to attend in person, we can put up your poster in the room. To enable this, please order it via the <a href="https://iclr.cc/Conferences/2025/PosterInstructions" rel="nofollow" target="_blank">official ICLR printing service</a> and contact <a href="mailto:chenmengyi@u.nus.edu">Mengyi Chen</a>. We can also arrange for you to upload the recording to SlidesLive, for this contact <a href="mailto:kna@nus.edu.sg">Nikita Kazeev</a>.</p>
